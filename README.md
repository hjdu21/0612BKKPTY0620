## ⚙️ 로컬에서 보기
1. 이 저장소를 클론합니다:
```bash
git clone https://github.com/hjdu2/0612BKKPTY0620.git
cd 0612BKKPTY0620
```

2. 브라우저에서 `index.html`을 열어주세요.

또는 로컬 서버를 실행합니다:

**Python 사용:**
```bash
# Python 3.x
python -m http.server 8000
# 그 후 브라우저에서 http://localhost:8000 열기
```

**Node.js 사용:**
```bash
# npm install -g http-server (처음 한 번만)
http-server
# 그 후 브라우저에서 표시된 주소 열기
```

**VS Code Live Server 사용:**
- Live Server 확장 설치 후 `index.html`을 우클릭 → "Open with Live Server"
